### 📋 Команды для запуска

```bash
# Установка зависимостей
pip install -r requirements-dev.txt

# Запуск всех тестов
pytest

# С подробным выводом
pytest -v

# С покрытием кода
pytest --cov=src --cov-report=html

# Просмотр HTML отчета
start htmlcov/index.html  # Windows
```
